# 금옥이 — Claude Code 데스크톱 마스코트 설계서

> 작성일: 2026-06-24
> 한 줄 요약: 화면 위에 떠 있는 회색 물개 **금옥이**가, Claude Code 세션의 상태(시작·작업중·끝·대기·종료)를 짧은 말풍선으로 알려주는 데스크톱 마스코트(데스크톱 펫).

---

## 1. 목적과 성공 기준

**목적** — 사용자가 여러 폴더에서 Claude Code를 돌릴 때, 지금 무슨 상태인지(시작했는지, 일하는 중인지, 멈춰서 기다리는지, 끝났는지)를 화면 한 구석의 귀여운 물개가 짧은 한마디로 알려준다.

**성공 기준**
- 어느 폴더에서 Claude Code를 켜든 금옥이가 상태 변화를 알아챈다 (전역 동작).
- 상태가 바뀌면 몇 초 안에 금옥이 표정·말풍선이 바뀐다.
- 금옥이가 꺼져 있어도 Claude Code는 전혀 영향받지 않는다 (반대도 마찬가지).
- 작업 중인 다른 창을 클릭으로 방해하지 않는다 (빈 공간 클릭 통과).

**이번 범위에서 제외 (YAGNI)**
- 여러 세션 동시 실행 시 세션별 개별 표시 → 일단 "가장 최근 이벤트" 하나만 표시.
- 컴퓨터 부팅 시 자동 실행 → 나중에 원하면 추가.
- 소리/효과음 → 제외.

## 2. 전체 구조

두 부분 + 중간 메모지로 분리한다. 직접 연결하지 않는 이유는 한쪽이 멈춰도 다른 쪽이 안 꼬이게 하기 위함.

```
①  Claude Code  ──(이벤트 발생 시)──▶  📝 상태 메모지(JSON 파일, 컴퓨터에 딱 하나)
                                              │
                                              ▼ (금옥이가 계속 지켜봄)
②  금옥이 앱(Electron) ── 메모지가 바뀌면 → 표정 변경 + 말풍선
```

- **① Claude Code 쪽:** hook(작은 연결고리)이 이벤트가 생길 때마다 메모지에 한 줄(JSON) 적는다. 기존 hook 설정은 그대로 두고 **추가만** 한다.
- **📝 메모지:** 전역 경로의 JSON 파일 하나. 위치: `C:\Users\User\.claude\geumoki\status.json`. 전역이라 어느 폴더 세션이든 같은 파일을 쓴다 → "어떤 세션이든 알아챈다"가 이걸로 해결됨.
- **② 금옥이 앱:** Electron 앱. 메모지를 감시(watch)하다가 바뀌면 화면의 금옥이를 갱신.

## 3. 금옥이 앱 (Electron)

### 3.1 창
- 프레임 없음(frameless) + 투명(transparent) + 항상 위(alwaysOnTop) + 작업표시줄 숨김(skipTaskbar).
- 작은 창에 금옥이 + 말풍선만 보임. 배경/테두리 없음 → 바탕화면에 물개만 얹힌 모습.
- 빈 영역 클릭은 뒤 창으로 통과(click-through), 금옥이 픽셀 위에서만 클릭 반응.

### 3.2 생김새
- 회색 동그란 몸통 + 빨간 목도리.
- **그림 파일을 쓰지 않고 코드로 픽셀을 찍어** 렌더 (canvas 픽셀아트). 외부 에셋 불필요.
- 상태별로 표정/포즈가 살짝 다름 (작업중=집중, 끝=만족, 대기=갸웃 등).

### 3.3 행동 ("가끔 어슬렁" + 물개 특성)
- 평소: 화면 구석에 얌전히 앉음.
- 가끔(수 분 간격, 랜덤): 옆으로 뒤뚱뒤뚱 살짝 이동 후 복귀, 통통 튀기.
- **물개다운 랜덤 행동:** 물개 박수👏, 생선 오물오물🐟, 데굴데굴/몸 비틀기, 물 튀기기💦"푸하—", 코로 공 굴리기.
- 상태와 살짝 연동: 작업 끝 → 박수 확률↑, 심심 → 뒹굴 확률↑.

### 3.4 사용자 조작
- 마우스로 잡아서 위치 이동.
- 우클릭 메뉴: **잠깐 숨기기 / 종료** (+ 나중에: 시작 시 자동 실행).

## 4. Claude Code 연결 (hooks)

전역 `~/.claude/settings.json`에 hook을 **추가**한다(기존 항목 유지). 각 hook은 Node 스크립트 한 개(`write-status.js`)를 호출해 메모지에 상태를 적는다 (Node는 이미 설치돼 있음).

| 금옥이 상태 | Claude Code hook 이벤트 | 비고 |
|------------|------------------------|------|
| 세션 시작 | `SessionStart` | |
| 작업 시작 | `UserPromptSubmit` | 사용자가 입력 보낸 순간 |
| 작업 끝 | `Stop` | 응답 완료 |
| 멈춤/대기 | `Notification` | 권한 요청·입력 대기 |
| 세션 종료 | `SessionEnd` | |
| 심심(잡담) | (hook 아님) | 앱이 "작업 끝난 뒤 일정 시간 조용함"을 감지해 idle 처리 |

메모지(JSON) 예시:
```json
{ "state": "working", "event": "UserPromptSubmit", "ts": 1719200000000 }
```

## 5. 말풍선 대사 (톤: 순딩순딩, 의심·걱정 없이 순순히 열심히)

대사는 **별도 파일**(`messages.js` 또는 `messages.json`)에 상태별 배열로 모은다. 코드를 몰라도 따옴표 안 글자만 바꿔서 추가/수정 가능.

| 상황 | 대사 예시 (각 상황 여러 개 중 랜덤) |
|------|-----------------------------------|
| 세션 시작 | "왔구나! 오늘도 시작해볼까~", "안녕! 나 깨어났어~" |
| 작업 시작 | "응! 할게~", "오케이~ 가보자~", "알겠어! 해볼게~" |
| 작업 중 | "하는 중이야~", "조물조물 하는 중🐟", "거의 다 됐어~", "열심히 하고 있어!" |
| 작업 끝 | "다 했어~ 헤헤", "짠— 끝!", "어때 어때?" (👏 박수) |
| 멈춤/대기 | "이거 확인 좀 해줘!", "기다리는 중...👀" (갸웃) |
| 세션 종료 | "수고했어, 잘 자~", "또 보자!" (지느러미 흔들기) |
| 심심할 때 | "심심해...", "물개는 자유다🌊" (뒹굴/물 튀기기) |

## 8. 안전/예외 처리

- 메모지 파일이 없거나 깨졌으면 금옥이는 조용히 기본(idle) 상태 유지 — 오류로 멈추지 않음.
- hook 스크립트는 실패해도 Claude Code 동작을 막지 않도록 빠르게 끝나고 조용히 종료(에러 무시).
- 금옥이 앱이 꺼져 있어도 메모지만 쌓일 뿐 문제 없음.
```
